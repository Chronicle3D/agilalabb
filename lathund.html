Git Cheat Sheet

  • git diff: Visar skillnaden mellan arbetskatalogen och det senaste commitade tillståndet.
  • git fetch <remote>: Hämtar alla ändringar från det angivna fjärranslutna repositoriet.
  • git push -u <fjärrnamn> <branch>: Skickar lokal branch till det fjärranslutna repositoriet och sätter en upstream-länk för tracking.
  • git pull <fjärrnamn> <branch>: Hämtar ändringar från det fjärranslutna repositoriet och merger dem in i den nuvarande branchen.
  • git checkout -b <nybranch>: Skapar och byter till en ny branch.
  • git clone --branch <branch> <url>: Klona ett specifikt branch från ett Git-repositorium till din lokala maskin.
  • git remote add <fjärrnamn> <url>: Lägger till en ny fjärranslutning med namnet <fjärrnamn> och URL:en <url>.
  • git rebase <branch>: Flyttar commits från en branch till en annan genom att ändra commitens baskommit.
  • git log --oneline: Visar commit-historiken i en komprimerad en-rad-format.
  • git rm <fil>: Tar bort en fil från arbetskatalogen och stage-området.
  • git tag <tagname>: Skapar en ny tagg för den senaste commiten.
  • git grep "<sökord>": Söker igenom alla filer i arbetskatalogen efter ett specifikt sökord.
  • git reset --soft HEAD~1: Återställer senaste commit, men behåller ändringarna i arbetskatalogen.
  • git cherry-pick <commit-id>: Applicerar en specifik commit till den aktuella branchen.
  • git bisect: Används för bisektion av historiken för att hitta vilken commit som introducerade en bugg.
